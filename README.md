# Pixel Adventure Game

A fully functional 2D adventure platformer game built with HTML, CSS, and JavaScript. The game is designed to work on both desktop and mobile browsers with responsive controls and colorful graphics.

## Features

- Player character that can run, jump, and climb
- Multiple levels (3) with increasing difficulty
- Obstacles and enemies to avoid
- Collectibles like coins and keys
- Score and lives system
- Attractive 2D graphics
- Keyboard controls for desktop (arrow keys + space)
- Touch controls for mobile devices
- Responsive layout that adapts to screen size

## How to Play

- **Desktop**: Use arrow keys to move and spacebar to jump
- **Mobile**: Use on-screen buttons to control the character
- Collect coins for points
- Grab keys to unlock doors
- Avoid enemies and obstacles
- Reach the door to advance to the next level

## Files

- `mobile-platformer-game.html`: Main game file containing all HTML, CSS, and JavaScript
- `index.html`: Redirect file for proper routing
- `vercel.json`: Configuration file for Vercel deployment

## Deployment

The game is configured for deployment on Vercel with proper routing and cache settings.
